# Driver Logbook — Design Plan

## App Concept
Android-приложение для водителей-профессионалов. Позволяет вести учёт путевых листов по дням, фиксировать маршруты, километраж, грузы и автоматически рассчитывать заработок.

---

## Screen List

### 1. Home Screen (Главный экран)
- Заголовок с текущим месяцем/годом и стрелками навигации
- Карточки статистики: заработок за день (выбранный) и за месяц
- Интерактивный календарь (сетка дней)
  - Дни с путевыми листами отмечены точкой/подсветкой
  - Тап по дню → открывает путевой лист
- Нижняя навигация: Главная | Настройки

### 2. Waybill Screen (Путевой лист)
- Заголовок: дата дня
- Поле: Номер путевого листа
- Выбор автомобиля (dropdown из настроек)
- Секция маршрутов (список, можно добавлять несколько):
  - Текстовое поле "Откуда → Куда" (свободный ввод)
  - Dropdown быстрого выбора маршрута из сохранённых
  - Переключатель: Гружёный / Пустой
  - Dropdown выбора груза (из настроек)
  - Поле: Кубатура / количество груза
  - Поле: Километраж гружёный
  - Поле: Километраж пустой
  - Кнопка удаления маршрута
- Кнопка "Добавить маршрут"
- Итого за день: расчёт заработка
- Кнопка "Сохранить"

### 3. Settings Screen (Настройки)
- Секция "Автомобили": список, добавить/удалить
- Секция "Маршруты": список сохранённых маршрутов, добавить/удалить
- Секция "Грузы и расценки": список грузов с ценой за единицу (руб/куб), добавить/удалить/редактировать

## Key User Flows

### Добавление путевого листа
1. Главный экран → тап на дату в календаре
2. Открывается экран путевого листа для этой даты
3. Заполнение: номер листа, авто, маршруты
4. Для каждого маршрута: откуда-куда, груз, кубатура, км
5. Приложение автоматически считает заработок
6. Нажать "Сохранить"
7. Возврат на главный экран — дата отмечена, заработок обновлён

### Настройка грузов и расценок
1. Главный экран → таб "Настройки"
2. Секция "Грузы и расценки" → "Добавить груз"
3. Ввести название (напр. "Щепа"), единицу (куб), цену (1.66 руб)
4. Сохранить → груз доступен в путевом листе

## Navigation Structure
- Bottom Tab Bar: 2 вкладки
  - `index` → Главный экран (календарь + статистика)
  - `settings` → Настройки
- Modal/Stack: Путевой лист (открывается поверх главного экрана)

## Data Models

### Vehicle (Автомобиль)
```ts
{ id: string; name: string; plate: string }
```

### Cargo (Груз)
```ts
{ id: string; name: string; unit: string; pricePerUnit: number }
```

### SavedRoute (Сохранённый маршрут)
```ts
{ id: string; name: string }
```

### RouteEntry (Запись маршрута в путевом листе)
```ts
{
  id: string;
  from: string;
  to: string;
  savedRouteId?: string;
  isLoaded: boolean;
  cargoId?: string;
  cargoVolume?: number;
  kmLoaded: number;
  kmEmpty: number;
}
```

### Waybill (Путевой лист)
```ts
{
  id: string;
  date: string; // YYYY-MM-DD
  number: string;
  vehicleId?: string;
  routes: RouteEntry[];
  totalEarnings: number;
}
```
